Introduction

Obsidian is a powerful knowledge base that works on top of a local folder of plain text Markdown files.

Key features that make Obsidian unique:

  • Local-first and plain text
  • Link-based note organization
  • Extensible with plugins
obsidian://open?vault=YourVaultName
Getting Started

To begin with Obsidian, first download and install the application.

Create your first vault by following these steps:

1. Click "Create new vault" 2. Choose a name and location 3. Select "Create"

Basic concepts to understand:

  • Vaults are folders on your computer
  • Notes are markdown files
Markdown Basics

Obsidian uses Markdown for formatting.

Here are some basic syntax examples:

# Heading 1 ## Heading 2 **bold** *italic* - bullet point

Advanced formatting options:

  • Tables
  • Code blocks
  • Blockquotes
Links and Backlinks

Links are a core feature of Obsidian.

Creating links between notes:

[[Note Name]] [[Note Name|Alias]]

Backlinks appear in the right sidebar by default.

  • Internal links
  • External links
  • Embedded notes
Plugins and Themes

Obsidian can be customized with plugins and themes.

Types of plugins available:

  • Core plugins
  • Community plugins
Settings → Community plugins → Browse

Theme customization:

You can modify the appearance using CSS snippets.

Advanced Features

Obsidian offers several advanced features that enhance your note-taking experience.

Some of the most powerful features include:

  • Command palette for quick actions
  • Custom hotkeys
  • Workspace layouts
  • Note templates

Using the command palette:

Press Ctrl/Cmd + P to open Type your command Press Enter to execute

Templates are particularly useful for standardizing your notes:

--- date: {{date}} tags: --- # {{title}}
Graph View

The Graph View is one of Obsidian's most distinctive features.

It provides a visual representation of your notes and their connections.

  • Local graph shows connections to current note
  • Global graph shows all vault connections
  • Customizable filters and groups

Graph view shortcuts:

Ctrl/Cmd + G : Open graph view Click and drag to navigate Scroll to zoom in/out

You can customize the graph appearance in settings:

Settings → Appearance → Graph View - Node size - Link thickness - Color schemes
Daily Notes

Daily notes help maintain a consistent journaling practice.

Setting up daily notes workflow:

  • Enable the Daily Notes core plugin
  • Configure template and location
  • Use hotkeys for quick access

Creating daily notes automatically:

1. Go to Settings 2. Enable Daily Notes plugin 3. Set template location 4. Configure format

You can also link to daily notes using special syntax:

[[{{date:YYYY-MM-DD}}]] [[{{date-7d:YYYY-MM-DD}}]] # Last week
